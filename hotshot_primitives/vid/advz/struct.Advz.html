pub struct Advz<P, T, H>where
    P: PolynomialCommitmentScheme,{ /* private fields */ }
Expand description

Context data for the ADVZ VID scheme.

This struct is a concrete impl for VidScheme. Generic parameters T, H are needed only to express trait bounds in the impl for VidScheme.

  • H is a hasher.
  • T is a group.
